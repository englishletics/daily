▶ 1. It's easy to learn English. É fácil aprender inglês.

  1. ▷ It’s hard to learn Japanese.
    É difícil aprender japonês.
  2. ▷ It’s necessary to do a good job.
    É necessário fazer um bom trabalho.
  3. ▷ It isn’t good to be lazy.
    Não é bom ser preguiçoso.
  4. ▷ It is wrong to go that way.
    É errado seguir assim.
  5. ▷ It’s important to have friends.
    É importante ter amigos.

▶ 2. Verb + '~ing' = it Verbo + '~ing' = it

  1. ▷ Learning how to cook is easy.
    Aprender a cozinhar é fácil.
  2. ▷ Saving money for the future is necessary.
    Poupar dinheiro para o futuro é necessário.
  3. ▷ Listening to music is fun.
    Ouvir música é divertido.
  4. ▷ Loving you is easy because you are beautiful.
    Amar você é fácil porque você é linda.

▶ 3. So and Such Variações de Tão

  1. ▷ That movie is so good. / It’s such a good movie.
    Esse filme é tão bom. / É um filme tão bom.
  2. ▷ The idea is so bad. / This is such a bad idea.
    A idéia é tão ruim. / É uma idéia tão ruim.
